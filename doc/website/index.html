Grappa

Grappa enables high-throughput graph processing on commodity clusters.

Learn more »

Exploit irregular parallelism

Big graph problems present big performance problems on commodity hardware. Finding locality is difficult, and parallelism is irregular, making it hard to balance across a cluster. Grappa helps you express and exploit the parallelism in your application.

View details »

Trade latency for throughput

Grappa's lightweight context switching lets that lead to better performance. Given lots of parallelism, Grappa can trade latency for high random-access memory bandwith, better effective network injection rates, and fine-grained lightweight synchronization.

View details »

Beat special-purpose hardware

Early results suggest that Grappa enables commodity hardware to crunch graphs faster than special-purpose supercomputers like the Cray XMT.
